Выдаст 10

5.
function a() {
    alert(this);
}
a.call(null);

Выдаст [object Window]

this в глобальной области указывает на window. Вызов call() у функции с параметром null равнозначно вызову call() без аргументов. А без аргументов вызываемая функция не «биндится» ни к какому объекту (или «биндится» к «объекту по умолчанию» — window).
